## What Is a Watermark in PDF?

A watermark is a semi-transparent text or image placed over a PDF document to indicate:

* ownership
* branding
* confidentiality
* copyright protection

### Example uses:

* “CONFIDENTIAL”
* “DRAFT”
* Company logo
* Author name
* Copyright text

Watermarks do not interfere with readability but clearly establish document identity.

Modern watermark tools allow users to:

* add text watermark
* insert image watermark
* adjust transparency
* control positioning
* apply across all pages
