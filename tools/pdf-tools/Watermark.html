The Art of Subtlety: Balancing Visibility and Readability

One of the many pitfalls of adding watermarks is making it too excessive. A watermark that is too intense, too dark, or too big will obscure the file so the recipient can't read the document or content. It will just be the kind that will anger the recipient and accomplish nothing for the sender. Finding the perfect watermark is all about balance–to watermark so that it is seen enough to be seen, but not so obscured that it reduces the content to nothing else.

JKM Edit's Add Watermark tool is made just for this. Being able to set the transparency of a watermark allows to make a logo discreet, so the text can be read over it.
